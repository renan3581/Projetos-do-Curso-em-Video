Formatações de texto

    
<strong> - Negrito.
<b> - Negrito com sentido de menos importancia. 
<em> - Itálico. 
<i> - Itálico com sentido de menos importancia. 
<mark> - Texto marcado. 
<small> - Texto pequeno.  
<del> - Texto deletado. 
<ins> - Texto inserido. 
<sup> - Texto sobrescrito. 
<sub> - Texto subrescrito. 
<code> - Letras com o mesmo espaçamento. 
<pre> -  Faz com que o texto fique exatamente como está escrito no editor, seja quebras de linha ou espaçamentos.
<q> - Coloca aspas na frase.
Blockquote – Coloca o texto com espaçamento para indicar que é uma citação.

Outros

Cite=“” - Faz com que o google e outros mecanismos de pesquisa reconheçam de onde você tirou essa citação.(link) 

<abbr> - Faz com que quando você coloque o mouse em cima de uma palavra abreviada o termo completo apareça. 
    
<bdo> - Texto invertido. (Utilidade = Nenhuma) 
    
<li> - Indicar que são itens de lista. 
    
<ol> - Indicar que são listas onde a ordem importa 
    
type=“” - Dentro de ol seleciona o tipo de contagem (1., I., A., Etc..) 
    
start=“” - Dentro do ol indica de qual número, letra a contagem deve começar. 
    
<ul> - Listas não ordenadas. 
    
.. = É usado para poder voltar na pasta anterior, possibilitando o uso de arquivos nela presente 

<a> - Coloca uma determinada palavra como um link - # > link falso
target="blank" - Faz com que seja aberto em uma nova aba.

Estilo

<style> 
     
     body {background-color: black; font-size: 20px; font-style: normal; font-variant: small-caps; color: grey; margin: 100px 80px 60px 40px;}

     a {color: cyan;}

A tag <a>, so pode ter sua cor alterada através de uma linha de codigo individual.
    
text-transform: lowercase - transforma todo o texto da parte selecionada em lowercase.

Não é necessario que você escreve uma linha de codigo para cada componente do body, você pode colocar-los em uma unica linha:
h1, h2, p, pre {color: cyan;}
    
</style>

Box-Model

    p{
        margin-top:50px
        margin-right:40px
        margin-bottom:30px
        margin-lef:20px
    }

Forma Abreviada:
p{margin: 50px 40px 30px 20px;}

Caso um valor não seja digitado, sera utilizado o mesmo valor do lado oposto;
Caso seja digitado apenas um valor, todos os lados recebem esse mesmo valor;
    
    p{
        width: 1400px  - Pode ser porcentagem tambem.
        margin: auto;
    }  

Desta forma o proprio navegador faz a divisão das margens, de acordo com a sobra de largura do monitor - Largura informada;
Caso o valor informado esteja em porcentagem, a diferença sera calculada de acordo com o tamanho da janela, e não com a do monitor

Borda - Border

    body{
        border-style: dotted;
        border-radius: 5px;
        border-color: rgb(250, 250, 250);
    }

Forma Abreviada:
body{border: dotted 5px rgb(250, 250, 250);}

Cada lado pode ser estilisado de forma individual.

    body{
        border-top-style
        border-right-width
        border-bottom-color
        border-lef-color
    }

Forma abreviada:
body{border-top:dotted 10px red;}

Links


Link 1 - Abre na mesma pagina.
Link 2 - Abre em outra pagina.

a:hover {color: red} - Faz com que quando você passe o mouse sobre o link, a cor dele mude.
a:hover {background-color: red} - Faz com que quando você passe o mouse sobre o link a cor de fundo do link mude.
a:visited {color: teal;} - Faz com que quando você ja tenha acessado o link ele fique com uma cor diferente.
a:active{color: black} - Faz com que quando você precione o mouse sobre o link ele fique com uma cor diferente, para indicar que você esta realmente clicando.
